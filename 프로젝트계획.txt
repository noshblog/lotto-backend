로또 생성기 프로젝트

1. 요구사항 정의
	- 전화번호+이름으로 간단한 회원가입 및 로그인 기능
	- 숫자 뽑기 기능
		- 무작위 6개 추출
		- 1개 이상 내가 원하는 번호 넣고, 나머지만 랜덤 추출
	- 내가 구매한 번호 등록/수정/삭제
	- 최신 회차 당첨번호 가져와서 비교
	- 과거 회차 당첨번호 저장 후 통계 분석
2. 개발환경 세팅
	- front
		react-native : 앱 개발
	- back
		spring-boot : 회원가입/로그인/기능 구현
		spring-batch : 최신 회차 정보 주기적으로 가져오기
3. github repository back/front 생성
4. 개발 진행
